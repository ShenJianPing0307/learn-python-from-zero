### （一）乔治布尔

乔治布尔，1847年，《逻辑的数学分析》，逻辑代数（布尔代数）。集合

>金属可以导电
>
>铅是金属
>
>因此，铅可以导电

```PYTHON
M # 所有金属集合
P # 所有导电的东西集合

M * P = M  1

S # 铅集合
S * M = S  2

S * M * P = S => S * P = S
```

引入了集合，引入了交集、并集..，另外引入了数字1和0表示命题的真假。

```python
A = "今天会下雨"
A = 1 # 真

B = "今天会出太阳"
B = 1/0
```
